# Customer Churn Analysis (Excel Project)

This project analyzes customer churn for a telecom company using Excel.  
It explores why customers leave, identifies the major drivers of churn, and visualizes key insights using pivot tables and charts.

---

## Project Objectives
- Understand the factors contributing to customer churn  
- Analyze customer behavior across contract types, payment methods, and tenure  
- Identify high-risk customer groups  
- Provide actionable recommendations to reduce churn
